空间优化
--------

滚动数组
________

以一个最简单的例子来说明。

累加求和用动态规划可以表示为 :code:`dp[i] = dp[i - 1] + i`。
可以发现 ``dp[i]`` 只依赖于 ``dp[i - 1]``，
所以我们可以用单个变量不断滚动，
最终代码就变成了循环求和。

斐波那契数列也类似，只不过 ``dp[i]`` 依赖于 ``dp[i - 1]`` 和 ``dp[i - 2]``，
所以我们需要两个变量来滚动。

01 背包
-------

常见的状态转移部分的代码为

.. code:: cpp
  if (j < weight[i]) {
    dp[i][j] = dp[i - 1][j];
  } else {
    dp[i][j] = std::max(dp[i][j], dp[i - 1][j - weight[i]] + value[i]);
  }

注意到可以用滚动数组优化空间，
但需要注意的是 ``dp[i][j]`` 会依赖于 ``dp[i - 1][j - wight[i]]``，
也就是说需要前一个状态下容量更小的背包来计算当前的状态，
所以如果使用滚动数组就必须从容量大的状态开始向容量小的状态更新。
